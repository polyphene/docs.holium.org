# `Usage scenario`

## Run via data CID

| Step                          | Command | 
| ----------------------------- | -------- | 
| Initialize project   | `holium init`       | 
| Import modules (loop) | `holium module import`       | 
| Connect modules (loop) | `holium connector add`       | 
| Import data (loop) | `holium data import`       | 
| Run using data CID | `holium run`       |
